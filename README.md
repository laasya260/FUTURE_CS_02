# Phishing Detection & Awareness Report

This project analyses phishing emails to help people recognise and avoid them. It includes three email samples, a full report explaining how each attack works, and simple tips for staying safe.

The three emails analysed :

The first is a fake account alert telling the user to verify their details immediately or get locked out. The second pretends to be the IT helpdesk saying a password is expiring. The third impersonates the CEO requesting an urgent bank transfer.

Tools used - Google Apps Toolbox and MXToolbox were used to check where emails actually came from.

Each email was checked for warning signs like fake sender addresses, urgency language, suspicious links, and requests for passwords or money. The links were inspected safely using online tools. Each email was then given a risk rating and the findings were written up in plain language anyone can understand.

**Disclaimer**

All email samples are reconstructed from real patterns for educational purposes only. No malware or attack tools are included.
